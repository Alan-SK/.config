## instructions
| Command    | Effect                                    |
|------------|-------------------------------------------|
| `h`        | Move to the left                          |
| `j`        | Move down the                             |
| `k`        | Move up                                   |
| `l`        | To the right                              |
| `H`        | Move to the beginning of the current line |
| `J`        | Move down five rows                       |
| `K`        | Move up five rows                         |
| `L`        | Moves to the end of the current row       |
| `(Ctrl+l)` | Open the Floaterm window                  |
